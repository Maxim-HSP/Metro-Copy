# Maintenance 系统维护中

## 使用

Maintenance 的宽高已经设为 100% ，会自动撑满可用空间。

当系统在维护中时，API网关返回的数据中 `code` 是 `3001` ，前端可以拦截 http 请求对这个状态码进行处理即可。

```js

  <Maintenance title='xxx系统维护中...' />
</div>
```
